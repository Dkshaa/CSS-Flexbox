# CSS Flexbox Demo

A small practice project that demonstrates how to build a simple navigation bar with CSS Flexbox.

## Files

- `index.html` contains the page structure.
- `flexbox.css` contains the layout and visual styles.

## What this project practices

- Creating a horizontal navigation layout.
- Using `display: flex` to align items.
- Spacing navigation links with `gap` and related flexbox properties.
- Styling a call-to-action button.

## How to view

Open `index.html` in a browser to see the navigation bar demo.
