# Flexbox Concepts

This project uses Flexbox to arrange the logo, navigation links, and contact button in one row.

## Key properties

- `display: flex` turns the container into a flex container.
- `justify-content: space-between` pushes the first and last items apart.
- `align-items: center` vertically centers items inside the navigation bar.
- `gap` creates consistent spacing between child elements.

## Useful experiment

Try changing `justify-content` to `center`, `flex-start`, or `space-around` in `flexbox.css` and refresh the page to compare the layout changes.
