# Accessibility Checklist

Use this checklist when improving the navigation demo.

- Keep link text clear and descriptive.
- Make sure text and background colors have strong contrast.
- Preserve keyboard access for links and buttons.
- Add visible focus styles when custom button or link styles are introduced.
- Use semantic HTML elements such as `nav`, `header`, and `main` in larger versions of this page.

## Next improvement idea

Wrap the navigation links in a `<nav>` element and add an accessible label if the page grows beyond a simple practice demo.
