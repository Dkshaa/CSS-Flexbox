# Real-Time Practice Guide

Use this project to practice improving a simple Flexbox navigation bar in small, meaningful steps.

## Practice goals

- Improve HTML structure with semantic elements such as `header` and `nav`.
- Fix CSS selector issues that affect the actual page layout.
- Add responsive styles so the navigation works on smaller screens.

## Suggested workflow

1. Open `index.html` and identify the main page sections.
2. Review `flexbox.css` and check whether each selector matches the HTML.
3. Resize the browser window and observe how the navigation changes.
4. Make one focused improvement at a time and commit each change with a clear message.

## Reflection questions

- Which Flexbox property controls horizontal spacing?
- Why should navigation links be placed inside a `nav` element?
- What changes when the layout switches from row to column on mobile screens?
